# Crypto App Ionic

Aplikasi mobile sederhana menggunakan Ionic Vue untuk menampilkan data cryptocurrency secara realtime menggunakan CoinLore API.

## Fitur

- Menampilkan Rank
- Menampilkan Nama Cryptocurrency
- Menampilkan Symbol
- Menampilkan Harga dalam USD
- Tombol Refresh Data

## Teknologi

- Ionic Framework
- Vue 3
- TypeScript
- Axios

## API

https://api.coinlore.net/api/tickers/

## Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/Hersan-12/crypto-app-ionic.git
```

2. Masuk ke folder project

```bash
cd crypto-app-ionic
```

3. Install dependency

```bash
npm install
```

4. Jalankan aplikasi

```bash
ionic serve
```
